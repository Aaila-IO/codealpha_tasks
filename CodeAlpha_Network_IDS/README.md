# CodeAlpha Network Intrusion Detection System

## Task Description
This task demonstrates the setup and testing of a Network-Based Intrusion Detection System (IDS) using Suricata on CentOS 7.

## Tools Used
- Suricata IDS
- Nmap
- CentOS 7
- VMware Workstation

## Configuration
- Interface monitored: ens33
- HOME_NET configured
- Custom local rules added
- Default Suricata rules enabled

## Detection Tests Performed
- ICMP Ping detection
- TCP SYN port scan detection using Nmap

## Evidence
- `fast.log` contains alerts generated by Suricata
- `nmap_scan.txt` shows the simulated attack
- Configuration and rule files included

## Conclusion
Suricata successfully detected suspicious network activity and logged alerts, confirming correct IDS deployment and operation.
